Task Description

Develop a Java application with a graphical user interface (Java Swing) that implements the game “Memory”.
The game board must consist of a 4x4 grid (16 buttons) containing 8 pairs of identical cards placed in random positions.
All cards are hidden at the beginning of the game. During gameplay, the user selects two cards consecutively, which are then revealed.
If the selected cards match, they remain visible; otherwise, they are hidden again after a short delay.
The game ends when all cards have been successfully revealed, and a victory message is displayed.
